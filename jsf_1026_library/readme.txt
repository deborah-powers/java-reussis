
Crud en Maven et Jsf.
Afficher et modifier une liste de livres.
La liste est stoquée dans une HashMap.

Outils nécéssaires
	Tomcat
	Maven

Le mettre en place
	maven install

Lancer le projet via la console
	Copier target/jsf_1026_library-1.0.war
	Coller dans path/to/apache-tomcat-9.0.12/webapps/
	Lancer tomcat (startup)
	http://localhost:8080/jsf_1026_library/books/bookList.xhtml
